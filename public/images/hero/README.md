# Immagini Hero Section

Inserisci qui le immagini di sfondo per le sezioni hero.

## Nome file richiesti:

1. **home.jpg** (o home.png) - Per la homepage
2. **pokemon.jpg** (o pokemon.png) - Per la pagina categoria Pokémon
3. **yugioh.jpg** (o yugioh.png) - Per la pagina categoria Yu-Gi-Oh!
4. **onepiece.jpg** (o onepiece.png) - Per la pagina categoria One Piece
5. **other.jpg** (o other.png) - Per la pagina categoria Altri prodotti

## Formato consigliato:

- **Formato**: JPG o PNG
- **Dimensioni**: 1920x1080px (Full HD) o superiore
- **Peso**: Ottimizza le immagini per web (< 500KB se possibile)
- **Stile**: Immagini che funzionano bene con overlay scuro/semi-trasparente per leggibilità del testo

## Note:

- Le immagini verranno applicate come sfondo con overlay scuro per garantire la leggibilità del testo
- Le immagini vengono caricate da: `/images/hero/[nome-file].jpg` o `.png`
- Il codice supporta automaticamente sia JPG che PNG
